# ASPICCExtractor

Extracts ICC camera color profiles from Corel AfterShot Pro binary.

## Usage

Extracting from Linux binary:

`asp-icc-extractor AfterShot`

Extracting from Windows binary:

`asp-icc-extractor AfterShot.exe`
